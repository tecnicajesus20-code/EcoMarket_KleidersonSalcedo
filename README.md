# EcoMarket_KleidersonSalcedo
Proyecto de Desarrollo web, que tiene como objetivo poner a prueba las habilidades recien aprendidas sobre HTML5 y CSS.

El sitio web que usare como referencia para poner productos es el que voy adjuntar a continuacion
"https://ecomarketcolombia.com/index.php/store/"
Se crearan 3 Categorias: Hogar / Aceites / Ropa 

Las fuentes que se usaron son "Catchy Mager" para descripciones, "Notable" para titulos y "Poppins" como fuente base. Quedaron guardadas en carpeta /fonts de la raiz del proyecto, estan los archivos ttf y uno otf, ademas de una carpeta de los ZIP.
